# Privacy Policy / 隐私政策

**MiniMax Monitor**  
Last updated: April 2026 / 最后更新：2026年4月

---

## English

MiniMax Monitor does not collect, store, transmit, or share any personal information.

**API Key**: Your MiniMax API Key is stored locally on your device using macOS UserDefaults. It is never transmitted to any server other than the official MiniMax API endpoints (`minimax.io` or `minimaxi.com`) solely for the purpose of querying your token usage.

**Network requests**: The app makes HTTPS requests directly to MiniMax's official API servers to retrieve token plan usage data. No data is sent to any third-party server.

**No analytics**: The app contains no analytics, tracking, or crash reporting SDKs.

**No account required**: The app does not create accounts or require sign-in beyond providing your MiniMax API Key.

If you have any questions, please contact us via [GitHub Issues](https://github.com/Mibslee/MiniMaxMonitor/issues).

## 中文

MiniMax Monitor 不收集、不存储、不传输、不共享任何个人信息。

**API Key**：您的 MiniMax API Key 仅通过 macOS UserDefaults 保存在本地设备上，除用于向 MiniMax 官方 API（`minimax.io` 或 `minimaxi.com`）查询 Token 用量外，不会传输至任何其他服务器。

**网络请求**：本应用仅通过 HTTPS 直接向 MiniMax 官方 API 服务器发起请求以获取套餐余量数据，不经过任何第三方服务器。

**无数据分析**：本应用不包含任何数据分析、追踪或崩溃上报 SDK。

**无需注册账号**：本应用无需注册账号，不收集任何身份信息。

如有疑问，请通过 [GitHub Issues](https://github.com/Mibslee/MiniMaxMonitor/issues) 联系我们。
